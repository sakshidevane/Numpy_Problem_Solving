# Numpy_Problem_Solving

## Overview

This repository contains my **NumPy Problem Solving** created as part of my Python learning practice. The notebook includes practical problems solved using NumPy arrays, indexing, slicing, conditions, and matrix operations. The goal of this file is to improve problem-solving skills while building a strong understanding of NumPy.

NumPy is an important Python library used for numerical computing, data analysis, and machine learning. This notebook focuses on applying NumPy concepts through real examples instead of only theory.

## File Included

* `numpy_exercise.ipynb` – Jupyter Notebook containing NumPy exercises and solutions.

## Topics Covered

The notebook includes practice on the following exercises:

### 1. Sudoku Validation Using NumPy

* Creating a 9x9 Sudoku matrix using arrays
* Checking row sums
* Checking column sums
* Validating 3x3 sub-matrices
* Using loops and conditional logic with NumPy

### 2. Student Data Analysis

A sample dataset is used with columns:

* Age
  n- Math Marks
* Science Marks

Exercises performed:

* Finding matrix shape
* Calculating average age of students
* Extracting marks column-wise
* Finding highest science marks
* Filtering students who scored above 90 in Math
* Increasing marks using vectorized operations
* Counting students younger than 19
* Calculating subject-wise averages
* Filtering students scoring at least 80 in both subjects
* Replacing low science marks with 0 using conditional indexing

### 3. NumPy Concepts Practiced

* Array creation
* Indexing and slicing
* Boolean masking
* Aggregation functions (`sum`, `mean`, `max`)
* Conditional updates
* Matrix traversal
* Efficient numerical operations

The purpose of this notebook is to practice solving beginner-to-intermediate level NumPy problems. It also acts as a revision resource for interviews, coding practice, and future data science learning.
